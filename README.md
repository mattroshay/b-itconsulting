A custom-built portfolio website for a freelance IT infrastructure consultant, built with Ruby on Rails and Bootstrap.

🛠 Stack:
	•	Ruby on Rails · SCSS · Bootstrap 5
	•	Cloudinary (for media) · PostgreSQL · Heroku
	•	JS for scroll-to-top and carousel logic

🚀 Features:
	•	Custom SCSS-based grid system (Figma-aligned)
	•	Auto-scrolling project carousel with infinite loop
	•	Parallax background on hero section
	•	Scroll-to-top button with animation
	•	Embedded video CTA + social media links
	•	Fully responsive layout across all screen sizes
	•	Optimized asset management via Cloudinary

🔗 Deployment:

Hosted on Heroku and linked to a custom domain
